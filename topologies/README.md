# Topology Starters

## Directory Naming Conventions

- `tf` - Terraform template
- `cf` - AWS Cloudformation template
- `arm` - Azure Resource Manager template
- `aws` - Specific to AWS Cloud
- `gcp` - Specific to GCP Cloud
- `az` - Specific to Azure Cloud
